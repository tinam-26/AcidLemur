# AcidLemur
Thank you for playing Cat-ch 'Em All, brought to you by
Sara Ashcraft, Sean Kelly, Christina Modica, and Sandra Wiktor
of Acid Lemur Studios!

The goal is simple: catch one of every type of cat.
You'll need to move around the map and solve puzzles to catch some of them.

Here are the controls:

W - move up
A - move left
S - move down
D - move right
E - interact with object
R - restart game
X - view achievements
arrow key up & down - switch options in cat interaction menu

Cheat codes:

1 - jump to room 1
2 - jump to room 2
3 - jump to room 3
